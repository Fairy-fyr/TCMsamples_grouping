在针对柴胡、当归、白术、甘草、生姜、川芎、肉桂、熟地黄这八种中药材的中药药理或数据分析实验中，我们拥有大量的两味药组合样本（共28种组合，336个样本），但实验需求却是基于五味药复方的（共56种复方）。为了模拟或准备这些复方实验数据，需要将基础的两味药样本，精准地投放到对应的复方文件夹中。
核心目标：
智能分发：将 总样本 文件夹中的 336 个两两组合样本，根据集合包含逻辑，复制到 中药数据集 下的 56 个五味药复方文件夹中。
数据冗余：确保每个复方文件夹拥有 120 个符合条件的样本（即该复方中包含的所有两味药组合）。
精准校验：提供一套机制，确保分发后的数据在数量和逻辑内容上完全正确，杜绝错放或漏放。
具体操作流程与细节如下：
1.使用python运行organize_herbs.py文件即可快速将336个样本按组别分类进入各个文件夹中；
2.使用python运行data_verify.py文件即可快速校验这些样本有没有被分类错误；
3.使用这两个py文件时需要确认文件夹名字准确
